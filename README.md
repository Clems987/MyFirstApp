# Tic Tac Toe - Online Multiplayer

## Single Player (vs Computer)
Open index.html in a browser or visit the GitHub Pages URL.

## Online Multiplayer
Requires a server. Deploy to a free service like [Glitch.com](https://glitch.com):

1. Go to glitch.com and create a new "glitch-hello-node" project
2. Delete the default files and upload:
   - server.js
   - public/multiplayer.html
   - package.json
3. The URL will be like `https://your-project.glitch.me/multiplayer.html`
4. Share the URL with friends to play together!

## How to Play Online
1. Click "Create Game" to get a room code
2. Share the room code with a friend
3. Friend enters the code and clicks "Join Game"
4. Play begins automatically when both join
